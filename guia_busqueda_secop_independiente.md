```
Tipo de proceso:
  ☑ Contratación directa          ← La más común para persona natural
  ☑ Mínima cuantía                ← También acepta personas naturales
  ☑ Selección abreviada           ← A veces acepta personas naturales

  ☑ Publicado                     ← Procesos abiertos

  ☑ Presentación de oferta        ← Que aún se pueda aplicar
```

### ❌ EXCLUIR (lo que NO te sirve)

```
Tipo de proceso a EVITAR:
  ✗ Contratación régimen especial  ← Suelen ser contratos ya adjudicados
  ✗ Licitación pública             ← Casi siempre exigen persona jurídica
  ✗ Concurso de méritos            ← Requiere empresa consultora

Estado a EVITAR:
  ✗ Proceso en evaluación          ← Ya cerró
  ✗ Adjudicado                     ← Ya tiene ganador
  ✗ Celebrado                      ← Ya se firmó contrato
  ✗ Liquidado                      ← Ya terminó
```

## 4️⃣ Checklist Rápida: ¿Este Proceso Me Sirve?

Antes de invertir tiempo en un proceso, verifica estos **7 puntos**:

### 🟢 Pasa (puedes aplicar)

- [ ] **1. ¿Acepta persona natural?**
  → Busca en los pliegos: "podrán participar personas naturales y/o jurídicas"
  → Si SOLO dice "persona jurídica" → ❌ NO aplica

- [ ] **2. ¿El proceso está abierto?**
  → La fecha de cierre/presentación de ofertas debe ser FUTURA
  → Si dice "días de tiempo transcurrido" → ❌ Ya cerró

- [ ] **3. ¿Es competitivo o solo publicidad?**
  → ¿Tiene cronograma con múltiples hitos? → ✅ Competitivo
  → ¿Tiene solo 1 documento (contrato firmado)? → ❌ Solo publicidad

- [ ] **4. ¿Puedes cumplir la experiencia?**
  → ¿Cuántos contratos anteriores piden?
  → ¿Cuánto deben sumar en valor?
  → ¿Aceptan experiencia como persona natural o solo como empresa?

- [ ] **5. ¿Tienes las certificaciones técnicas?**
  → ¿Piden títulos profesionales específicos?
  → ¿Piden certificaciones (ISTQB, AWS, etc.)?

- [ ] **6. ¿NO exige RUP obligatorio?**
  → En contratación directa y mínima cuantía generalmente NO piden RUP
  → Si lo exigen y no lo tienes → ❌ No aplica (por ahora)

- [ ] **7. ¿El valor es razonable para persona natural?**
  → Contratos < $80M COP son típicos para persona natural
  → Contratos > $100M COP casi siempre piden persona jurídica

## 7️⃣ Señales de Alerta (Red Flags) — Procesos que NO te sirven

| 🚩 Señal | Significado |
|---|---|
| Solo dice "persona jurídica" | No acepta independientes |
| Pide "Certificado Existencia y Representación Legal" | Solo para empresas |
| Exige "SG-SST de la empresa" | Requiere persona jurídica |
| Pide "Revisor Fiscal" | Solo empresas grandes |
| Contrato ya firmado en documentos | Solo publicidad |
| Régimen especial + 1 solo documento | Ya adjudicado |
| Exige constitución UT/Consorcio obligatorio | No es para independientes |
| Pide garantía de "pago de salarios" | Implica que esperan empresa con empleados |
| Valor > $150M COP en contratación directa | Probablemente para empresa |

## 📱 Rutina Sugerida de Búsqueda

### Diaria (10 min)
1. Entra a SECOP II → Búsqueda de procesos
2. Filtra por códigos UNSPSC: `81111506`, `81111508`, `81111504`
3. Estado: Publicado
4. Revisa los nuevos del día

### Semanal (30 min)
1. Busca por palabras clave: "pruebas de software", "inteligencia artificial", "calidad de software"
2. Revisa procesos de **contratación directa** publicados en la semana
3. Aplica la checklist rápida a los que te interesen

> [!TIP]
> **Contratación directa** es donde más oportunidades hay para persona natural. Busca procesos con títulos como:
> - "Prestación de servicios profesionales de apoyo..."
> - "Prestación de servicios como..."
> - "Contrato de prestación de servicios profesionales..."
> Estos son los típicos contratos OPS (Orden de Prestación de Servicios) del Estado.
